EC AntiSpam bot для мессенджера Telegram будет автоматически определять и блокировать мультимедийные сообщения и ссылки от недавно вступивших в супергруппу участников, а также ряд известных спам-ботов.

Для работы данного программного продукта сначала вы должны получить API-токен у @BotFather. Он будет запрошен у вас на заключительном шаге установки.

EC AntiSpam bot поддерживает следующие способы хранения API-токена:

  * переменные окружения (токен будет сохранён в переменных окружения текущего пользователя системы) — используется по умолчанию;
  * скрипт запуска (токен будет сохранён непосредственно внутри автоматически сгенерированного скрипта) — устаревший, не рекомендуется.

Чтобы начать пользоваться ботом, сразу по окончании установки просто запустите его и при помощи Telegram-клиента добавьте в качестве администратора в каждую группу, которую хотите защитить.

Внимание! Для корректной работы бота функция скрытия участников (Hide Members) должна быть отключена!

Настоятельно рекомендуем вам ознакомиться со справочной системой по доступным командам:
https://github.com/xvitaly/ecasbot/blob/master/docs/available-bot-actions.md

При необходимости вы также можете изменять настройки простым редактированием файла конфигурации %APPDATA%\ecasbot\ecasbot.json в любом текстовом редакторе:
https://github.com/xvitaly/ecasbot/blob/master/docs/schema-documentation.md

Для вступления изменений в силу требуется перезапуск бота.
